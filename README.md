# 🧠 Web Development Quiz App

An interactive and responsive Quiz Application developed using HTML, CSS, and JavaScript. The app presents 10 multiple-choice questions related to Web Development and provides instant feedback, score tracking, and a restart option.

---

## 🎯 Objective

The objective of this project is to build a mini interactive JavaScript application using DOM manipulation. This project was developed as part of my Frontend Developer Internship task to strengthen my understanding of JavaScript, event handling, and dynamic UI updates.

## ✨ Features

- ✅ 10 Web Development multiple-choice questions
- ✅ Displays question counter (1/10, 2/10, etc.)
- ✅ Instant answer validation
- ✅ Highlights correct and incorrect answers
- ✅ Automatically reveals the correct answer
- ✅ Score calculation and final result display
- ✅ Restart Quiz functionality
- ✅ Fully responsive design
- ✅ Attractive gradient background and modern UI
- ✅ Footer with developer credit
